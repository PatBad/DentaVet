Features

Everything your dental workflow needs

Built by a veterinarian, for veterinarians. Every feature exists because it was needed in the clinic.

🦷

Modified Triadan System

Full 42-tooth charting for dogs and cats. Incisors, canines, premolars, and molars — each rendered with accurate crown and root anatomy.

⚡

One-click EzyVet sync

Push your completed chart directly into EzyVet's dental record fields with a single click. No copy-pasting. No re-entering data.

🔒

100% local — no cloud

All chart data is stored on your device using Chrome's local storage. Nothing is ever transmitted to any external server.

📋

AVDC condition codes

Periodontal disease staging, tooth resorption, fractures, mobility grading, furcation involvement, and more — all coded to AVDC standards.

📏

Probing depth recording

Record six-point probing depths per tooth (MB, B, DB, ML, L, DL) with furcation and mobility grading in a clean, fast interface.

📄

Formatted reports

Generate a structured dental findings report ready for the patient's medical record — formatted for medico-legal documentation standards.

How it works

Up and running in minutes

No setup, no configuration, no separate account. Install and start charting.

1

Install from Chrome Web Store

Add DentaVet to Chrome in one click. A free 30-day trial starts immediately — no credit card needed.

2

Open a patient in EzyVet

Navigate to a patient's dental record in EzyVet, then open the DentaVet side panel from the Chrome toolbar.

3

Chart teeth and record findings

Click or tap teeth to record findings. Use AVDC codes for conditions, probing depths, fracture types, and more.

4

Push to EzyVet

Hit the sync button — DentaVet fills all EzyVet dental fields automatically. Done. Ready for the next patient.
